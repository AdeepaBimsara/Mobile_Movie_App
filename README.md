
<h1 align="center">🎬 Movie Hub</h1>

<p align="center">
A modern and responsive Movie Discovery Mobile Application built with
<strong>React Native</strong>, <strong>Expo</strong>, <strong>TypeScript</strong>, and
<strong>Appwrite</strong>.
</p>

<hr>

<h2>📱 Overview</h2>

<p>
Movie Hub is a modern movie discovery application that allows users to explore
the latest trending movies, search for their favorite films, save movies to a
personal collection, and manage their profile through secure authentication.
The application provides a smooth and user-friendly experience with a clean UI
and real-time database integration using Appwrite.
</p>

<hr>

<ul>

<h2>🔑 Authentication</h2>

<p>
Movie Hub uses <strong>Appwrite Authentication</strong> to provide secure user
registration and login functionality.
Each user has their own account, allowing them to save and manage their favorite
movies securely.
</p>

<hr>

<h2>🗄 Database</h2>

<p>
The application uses <strong>Appwrite Database</strong> to store users'
favorite movies. Every authenticated user has their own personalized saved movie
collection.
</p>

<hr>

<h2>🎥 Movie Search</h2>

<p>
Users can search for any movie using the search bar. The application retrieves
movie information from the TMDB API and displays matching results instantly.
</p>

<hr>

<h2>❤️ Saved Movies</h2>

<ul>
<li>Save favorite movies.</li>
<li>View all saved movies.</li>
<li>Remove movies from the saved collection.</li>
<li>Personalized movie library for every user.</li>
</ul>

<hr>

<h2>🏠 Home Screen</h2>

<p>
The Home page displays a list of trending movies along with the latest movie
recommendations fetched from the TMDB API.
